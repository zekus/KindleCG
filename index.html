KindleCG

Directory Tree to Kindle Collections generator

A command line ruby script that will generate collections in your kindle based on the directory structure of your ebooks. The library to understand the mobi format are a straight port/refactor of the python plugin for calibre.

Installation

$ gem install KindleCG

Usage

After installing the gem you will have access to a command line tool kindlecg. Just run it and will generate collections based on the directory structure of your ebooks in your kindle device.

WARNING The script is under development. It override your current collections!!!
